## 📌 Overview

The **Student Course Allotment Management System** is a Java-based application with a graphical user interface built using **Java Swing**.  
It helps manage student registrations and efficiently assign courses while maintaining structured records.

---

## ✨ Features

- 📝 Student Registration  
- 🔢 Automatic Roll Number Generation  
- 📋 View All Students  
- 📚 Course Assignment  
- 🔍 View Student Course Details  
- 🖥️ User-friendly GUI (Java Swing)  

---

## 🛠️ Tech Stack

- **Language:** Java ☕  
- **GUI Framework:** Java Swing 🖥️  
- **Concepts:** Object-Oriented Programming (OOP), Collections Framework

1. Register a student  
2. System generates a unique roll number  
3. View all registered students  
4. Assign courses  
5. Display course allotment
